# Crimson-Period-Tracker
Crimson | Period & PCOS Tracker
Crimson is a holistic health management web application designed to help individuals track their menstrual cycles, manage PCOS symptoms, and gain deep insights into their hormonal health. Built with a focus on user privacy and actionable health data, Crimson provides personalized tips, nutrition guides, and fertility tracking.

🌐 Live Demo
Check out the application here: https://pokedex-oe7g.vercel.app/

✨ Key Features
📅 Cycle Tracking & Analytics
Smart Logging: Easily log period start dates, cycle length, and flow intensity.

Phase Prediction: Automatically calculates your current cycle phase (Menstrual, Follicular, Ovulatory, or Luteal).

Advanced Analytics: Visualize average cycle length, period duration, and temperature fluctuations over time.

🌸 PCOS Management
Daily PCOS Tips: Receive curated advice on hormonal regulation and insulin sensitivity.

Habit Tracker: Stay on top of essential habits like hydration, movement, sleep, and low-sugar intake.

Nutrition Guide: Access a comprehensive list of foods to enjoy or avoid, including specific Indian recipes like Methi Paratha and Flaxseed Ladoo.

🌡️ Fertility & Health Logs
Symptom Tracking: Log daily symptoms such as cramps, bloating, acne, mood swings, and basal body temperature (BBT).

Fertility Window: View your predicted ovulation day and "Best Conception Days."

Cervical Mucus Monitoring: Track changes to better understand your fertile window.

📊 Medical Metrics
BMI & Weight Tracker: Monitor your physical metrics and Waist-to-Hip ratio.

Blood Test Tracker: Keep a record of essential lab results including HbA1c, Vitamin D, and TSH levels with reference ranges.

📚 Education & Resources
Resource Hub: Find recommended reading, specialist contacts, and emergency helplines.

Educational Content: Integrated video guides on understanding cycles and PCOS nutrition.

🚀 Technologies Used
Frontend: React / Next.js (likely, based on the Vercel deployment)

Styling: Tailwind CSS / Styled Components

Deployment: Vercel

🛠️ Installation & Setup (For Developers)
To run this project locally:

Clone the repository:

Bash
git clone https://github.com/your-username/crimson-tracker.git
Navigate to the project directory:

Bash
cd crimson-tracker
Install dependencies:

Bash
npm install
Run the development server:

Bash
npm run dev
Open http://localhost:3000 in your browser.
